# Adonis API With ElasticSearc and Kibana

This test application for study with elastic search and kibana

1. Bodyparser
2. Authentication
3. CORS
4. Lucid ORM
5. Migrations and seeds
6. Elastic Search
7. Kibana

## Setup

```bash
yarn or npm install
cp .env.example .env
adonis key:generate
config auth database .env
adonis migration:run
adonis serve --dev
```
